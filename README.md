🛡️ Deep Work Guardian
A Chrome Extension built to eliminate "Productive Procrastination" by hard-blocking Reddit during focus sessions.

Key Features:

State Persistence: Uses chrome.storage.local to remember protection settings.

Dynamic Injection: Handles script injection into pre-existing tabs using chrome.scripting.

Zero-Refresh Updates: Real-time messaging between popup and content scripts.

Tech Stack:

JavaScript (ES6+), HTML5, CSS3, Chrome Extension API (Manifest V3).

-----------------------------------------HOW TO INSTALL-----------------------------------------

1)Download the ZIP from GitHub.

2)Extract the files in a Folder.

2)Go to chrome://extensions.

3)Enable "Developer Mode."

4)Click "Load Unpacked."

5)Select the folder from step 2.

6)In the extension details tab select 'pin to toolbar'.

7)Toggle ON or OFF for blocking.
